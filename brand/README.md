# Brand assets

Source logos for the provcheck app icon + the in-app branding.

- `cm_gh_white.png` — original Creative Mayhem shattering-lightbulb
  mark, black-on-white (what the Creative Mayhem GitHub profile
  uses). Source of truth.
- `cm_logo_transparent.png` — white-on-transparent variant, used for
  the in-app top-bar logo and as the source for the generated app
  icon set (`app/src-tauri/icons/*`). Regenerated from the original
  via a pixel-level inversion (see commit history).

The app icon set was generated by running

    npx tauri icon brand/cm_logo_transparent.png \
      -o app/src-tauri/icons

Re-run that if the logo changes.
